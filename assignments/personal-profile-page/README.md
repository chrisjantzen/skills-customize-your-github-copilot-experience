# 📘 Assignment: Personal Profile Page

## 🎯 Objective

Create a simple personal profile webpage to practice using HTML structure, basic CSS styling, and page layout.

## 📝 Tasks

### 🛠️ Build the Page Structure

#### Description
Create a webpage that introduces you with a title, short bio, and a few sections of content.

#### Requirements
Completed program should:

- Use semantic HTML elements such as `header`, `main`, and `footer`
- Include a heading and at least one paragraph of text
- Add at least one image or placeholder image
- Organize the content into clear sections

### 🛠️ Style the Page

#### Description
Apply CSS so the page looks more polished and visually organized.

#### Requirements
Completed program should:

- Use an external stylesheet or embedded CSS
- Change the background color or text color
- Adjust spacing with margins or padding
- Style headings and links so they are easy to read

### 🛠️ Add a Personal Touch

#### Description
Make the page feel more personal by adding details that reflect your interests or goals.

#### Requirements
Completed program should:

- Include at least three personal facts or interests
- Add a list, quote, or contact section
- Make sure the page is easy to read on a desktop browser
