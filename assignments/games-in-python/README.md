# 📘 Assignment: Games in Python

## 🎯 Objective

Build a text-based Hangman game in Python to practice working with strings, loops, conditionals, and user input.

## 📝 Tasks

### 🛠️ Create the Game Logic

#### Description
Implement a simple Hangman game where the player guesses letters to reveal a hidden word before running out of attempts.

#### Requirements
Completed program should:

- Randomly choose a word from a predefined list
- Show the word as underscores while letters are still hidden
- Accept single-letter guesses from the player
- Track the number of incorrect guesses remaining
- End the game when the word is guessed correctly or the player runs out of attempts
- Display a clear win or lose message at the end

### 🛠️ Improve the User Experience

#### Description
Add a few small features to make the game easier to play and more polished.

#### Requirements
Completed program should:

- Show letters that have already been guessed
- Inform the player when a guessed letter is not in the word
- Prevent repeated guesses from counting as extra turns
- Use clear prompts and messages so the game is easy to follow
